### Project Orchestrator Setup

Follow these steps to set up your project environment and run the application:

1. :
   ```sh
   python -m venv orchestrator
   ```

2. :
   ```sh
   source orchestrator/bin/activate
   ```

3. :
   ```sh
   pip install -r requirements.txt
   ```

4. :
   ```sh
   python ./run.py
   ```
